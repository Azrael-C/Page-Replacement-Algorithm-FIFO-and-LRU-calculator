# Page-Replacement-Algorithm-FIFO-and-LRU-calculator
A simple calculator that simulates **FIFO (First-In, First-Out)** and **LRU (Least Recently Used)** page replacement algorithms. It allows users to input reference strings and frame sizes to visualize page faults and understand how different memory management strategies work in operating systems.
